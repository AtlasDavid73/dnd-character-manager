About Dragon's Codex

Dragon's Codex is a comprehensive web application designed to help Dungeons & Dragons players and Dungeon Masters quickly access essential game information. Built using the D&D 5th Edition API, our platform provides an intuitive and visually appealing way to explore the vast world of D&D.

This project showcases modern web development techniques including responsive design, API integration, and interactive user interfaces. Whether you're preparing for your next session or just curious about the game, we've made it easy to find what you need.
